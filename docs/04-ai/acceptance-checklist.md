# 验收清单

用于每次文档更新后的最小验收，保证文档质量基线。

## 内容验收

- [ ] 标题与导航标题一致或通过 frontmatter 显式声明
- [ ] 无明显的错别字或格式错误
- [ ] 所有内部链接可正确解析
- [ ] Markdown 代码块语法正确
- [ ] 术语使用与项目约定一致

## 结构验收

- [ ] 文档位于正确的目录分区
- [ ] 已在导航配置中添加对应条目
- [ ] 搜索索引已包含新文档
- [ ] 上一篇/下一篇翻页路径连贯

## AI 可读验收

- [ ] 每个章节有明确的 `##` 标题
- [ ] 段落可用概念名称定位（非纯叙述）
- [ ] 关键数据（接口、参数、规则）以结构形式呈现
- [ ] 无模糊指代词（"这个"、"那个"、"上文所述"）

## 一致性验收

- [ ] 与其他文档无术语或流程冲突
- [ ] `rundoc check` 通过，无新增不一致项
- [ ] 引用的文件/路径/接口名称真实存在
