# 归档

存放已失效或过期的历史版本文档，保留记录但不参与当前导航。

## 归档标准

文档满足以下任一条件时移入本目录：

- 对应功能已下线或不再维护
- 对应流程已被新方案替代
- 超过 6 个月未更新且经确认不再需要
- 决策记录已被后续决策覆盖

## 归档流程

1. 在文档顶部添加归档标记：`<!-- archived: YYYY-MM-DD -->`
2. 将文件移入 `docs/07-archive/`
3. 从导航配置中移除对应条目
4. 在本次扫描报告中注明归档操作

## 注意事项

归档文档仍保留在 Git 历史中，可随时恢复。需要恢复时，移除归档标记并移回对应目录即可。
