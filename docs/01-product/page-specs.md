# 页面规格

记录产品页面行为与边界，随代码变更持续更新。

## 首页（`/`）

- **路由**：`/`
- **用途**：产品入口，展示 RunDoc 能力与导航入口
- **内容区块**：
  - Hero 区：一句话定义 + 核心行动按钮
  - 能力面板：RunDoc 能做什么
  - 角色入口：按用户角色组织阅读路径
  - 文档分区：按导航分类展示文档链接

## 文档页（`/docs/*`）

- **路由**：`/docs/**`
- **用途**：渲染 Markdown 文档内容
- **布局**：左侧导航 + 中间文档正文 + 右侧目录（响应式隐藏）
- **组件**：
  - 文档正文区：标题、摘要、正文内容
  - 目录组件：自动提取 h2/h3 标题，滚动高亮
  - 翻页导航：上一篇/下一篇文档链接

## 页面交互

- 搜索快捷键：`⌘K` 全局搜索
- 侧边栏：移动端抽屉模式，点击遮罩关闭
- 语言切换：支持中文/英文切换
- 响应式：980px 以下隐藏右侧目录，720px 以下隐藏顶部导航
